word-cloud
==========

About
-----

Create pretty visual word clouds using
[PyTagCloud](https://github.com/atizo/PyTagCloud) from a Facebook message history.

Installation
------------

Install PyTagCloud using pip:

        $ pip install -U pytagcloud

Or use easy_install:

       $ easy_install -U pytagcloud
