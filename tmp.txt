inbox is a dict containing 3 keys: paging, data, summary

1) paging is a dict containing 2 keys: next and previous both of which
are urls to get the next and previous entries to the inbox

2) data is a list. each element of data contains:
                   
                   a) comments - a dict containing paging and
                   data. data is a list containing each individual
                   message.each message is stored as a dict.
                   b) updated_time
                   c) to
                   d) unseen
                   e) unread
                   f) id
